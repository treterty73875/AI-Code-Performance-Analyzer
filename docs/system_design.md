#System Design

 ##Project Name
  AI-Powered Code Performance Analysis and Optimizations System

 ##Problem Statement
  Many programmers wirte code without knowing whether it is efficient.
  Analyzing time complexity, memory usage and optimization opportunities manually is difficult and time consuming.
  Our system will automatically analyze system code, identify performance bottlenecks, and provide AI-powered optimization suggestions.

 ##Objective
  Build an intelligent system that analyzes source code and help developers improved code quality and performance.

 ## Features
 1. upload a C++ code files.
 2. Analyze the source code.
 3. Estimate time complexity.
 4. Estiimate space complexity.
 5. Detect code smells.
 6. Detects performance bottlenecks.
 7. Suggest code optimizations using AI.
 8. Generate a Performance report.
 9. Display analysis results in a user-friendly interface.
